# WordPress Starter Theme

A clean, responsive WordPress starter theme with modern features and best practices. This theme is designed to be a solid foundation for building custom WordPress websites.

## Features

- **Responsive Design**: Mobile-first approach with fluid layouts
- **Clean Code**: Well-structured, documented, and follows WordPress coding standards
- **Modern CSS**: Uses modern CSS practices with flexbox layouts
- **SEO Friendly**: Semantic HTML5 markup
- **Translation Ready**: All strings are internationalized
- **Custom Logo Support**: Upload and display your own logo
- **Navigation Menus**: Custom menu support with fallback
- **Featured Images**: Full support for post thumbnails
- **HTML5 Support**: Modern markup for search forms, comments, galleries, etc.
- **Widget Ready**: Customizable widget areas (can be extended)

## Installation

1. Download the theme files from this repository
2. Upload the theme folder to `/wp-content/themes/` directory
3. Activate the theme through the 'Appearance > Themes' menu in WordPress
4. Configure theme settings through the WordPress Customizer

## Theme Structure

```
wordpress-starter-theme/
├── style.css          # Main stylesheet with theme information
├── index.php          # Main template file
├── header.php         # Header template
├── footer.php         # Footer template
├── functions.php      # Theme functions and features
├── single.php         # Single post template
├── page.php           # Page template
└── README.md          # This file
```

## Customization

### Navigation Menus

1. Go to **Appearance > Menus**
2. Create a new menu
3. Assign it to the "Primary Menu" location

### Custom Logo

1. Go to **Appearance > Customize**
2. Navigate to **Site Identity**
3. Upload your logo

### Colors and Styling

Edit `style.css` to customize colors, fonts, and other styling options.

## Template Hierarchy

- `index.php` - Default template for all content
- `single.php` - Template for single posts
- `page.php` - Template for pages
